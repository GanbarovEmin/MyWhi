# MyWhi — Native macOS app (Wispr Flow alternative)

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Переписать `Hermes Dictate` из «menu bar MVP» в полноценное native macOS приложение **`MyWhi`** с desktop-окном (sidebar + Insights + Scratchpad с markdown-редактором) и menu bar companion, разделяющими общий UI/движок. Сохранить существующую `transcribe.py` (faster-whisper) как **fallback**, primary engine — нативный **WhisperKit** (Argmax, Swift, в 2-5x быстрее на Apple Silicon).

**Architecture:** Один Swift `.app` бандл с двумя сценами — `WindowGroup` (desktop) и `NSStatusItem` (menu bar). Один `AppState` (singleton) — общий источник истины. Хранение: **Markdown vault** (`vault/YYYY/MM/YYYY-MM-DD-HHMMSS-<slug>.md` с YAML frontmatter) + **SQLite FTS5** индекс. Дизайн: SwiftUI-реализация Cohere tokens (palette, spacing, typography, components).

**Tech Stack:**
- **Swift 5.9+, SwiftUI, AppKit, AVFoundation** (есть)
- **macOS 14+** target
- **WhisperKit 1.0+** (`https://github.com/argmaxinc/argmax-oss-swift`, SPM) — primary engine
- **faster-whisper** (Python venv) — fallback engine (legacy)
- **SQLite** (built-in `libsqlite3`, FTS5 если доступно)
- **Markdown**: native `TextEditor` + preview (no deps)

## Risks & tradeoffs

| Риск | Митигация |
|---|---|
| **WhisperKit не скомпилируется** (Metal/CoreML binding issues на macOS 26) | Phase 0.8 verify build. Если fail — fallback на faster-whisper primary. |
| **WhisperKit model download медленный** (~1.5GB) | UI progress, можно начать с `small` модели (~500MB), upgrade в Settings. |
| **Миграция history.json теряет данные** | Бэкап `history.json.migrated-<date>`. Roundtrip test перед merge. |
| **Two scenes конфликтуют за AppState** | AppState singleton, ownership у AppDelegate. WindowGroup через `.environmentObject`. |
| **WindowGroup не открывается из NSStatusItem button action** | `@Environment(\.openWindow)` через NSHostingController wrapper. |
| **SQLite FTS5 не встроена** | Проверка `sqlite3_compileoption_get("ENABLE_FTS5")` на старте. Fallback на LIKE-search. |
| **CohereText font proprietary** | Fallback на SF Pro / system default. Документировать в коде. |
| **Day streak edge case (TZ)** | `Calendar.current.startOfDay` — пользователь в Asia/Baku, стабильно. |
| **Large vault (1000+ файлов) замедляет загрузку** | `VaultStore.listAll()` async в Task.detached, кэш в IndexStore. |
| **Cmd+Option+D конфликт** | Проверить существующие shortcuts; если занят — fallback на Cmd+Option+W. |
| **WhisperKit vs Python drift в результате** | EngineManager выбирает один источник истины, тесты с одинаковым synthetic audio. |
