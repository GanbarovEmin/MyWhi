# Hermes Dictate v2 — Native macOS app (Wispr Flow alternative)

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Переписать `Hermes Dictate` из «menu bar MVP» в полноценное native macOS приложение с desktop-окном (sidebar + Insights + Scratchpad с markdown-редактором) и menu bar companion, разделяющими общий UI/движок. Сохранить существующую `transcribe.py` (faster-whisper) и расширять инкрементально.

**Architecture:** Один Swift `.app` бандл с двумя сценами — `WindowGroup` (desktop) и `NSStatusItem` (menu bar). Один `AppState` (singleton) — общий источник истины. Хранение переезжает с `history.json` (10 записей) на **Markdown vault** + **SQLite индекс** для поиска/статистики. Дизайн-система на основе Cohere tokens (белый canvas, pill CTAs, monospaced-feel display, 22px media cards), реализована в SwiftUI.

**Tech Stack:**
- **Swift 5.9+, SwiftUI, AppKit, AVFoundation** (уже есть)
- **macOS 14+** (target уже 14, остаётся)
- **SQLite** через `sqlite3` (built-in, no deps) для индекса
- **Markdown UI**: встроенный `TextEditor` + naive parser для preview (без зависимостей)
- **faster-whisper** (Python venv) — оставляем, не трогаем

## Risks & tradeoffs

| Риск | Митигация |
|---|---|
| **Миграция history.json теряет данные** | Бэкап `history.json.migrated-<date>` перед переносом. Тест roundtrip перед merge. |
| **Two scenes конфликтуют за AppState** | AppState — singleton, ownership у AppDelegate. WindowGroup получает через `.environmentObject`. |
| **NSStatusItem + WindowGroup race condition** | AppSceneRouter сериализует через MainActor. Тест: открыть/закрыть 10 раз подряд. |
| **faster-whisper медленнее WhisperKit** | Оставляем faster-whisper (уже работает, пользователь доволен). WhisperKit — отдельная фаза при желании. |
| **SQLite FTS5 не встроена в macOS** | В macOS 14+ FTS5 есть в системной libsqlite3. Проверить `sqlite3_compileoption_get("ENABLE_FTS5")` на старте, fallback на LIKE-поиск. |
| **WindowGroup не открывается из NSStatusItem button action** | Использовать `@Environment(\.openWindow)` через NSHostingController wrapper. |
| **Cohere design system использует proprietary CohereText** | Fallback на SF Pro / system default. Документировать в коде. |
| **Day streak edge case: разные timezone** | Считать по `Calendar.current.startOfDay(for: date)` — пользователь в Asia/Baku, всё стабильно. |
| **Large vault (1000+ файлов) замедляет загрузку** | `VaultStore.listAll()` — async, в `Task.detached`, на первом запуске показывать progress, кэшировать в IndexStore. |

## Открытые вопросы для пользователя (можно решить перед стартом или в процессе)

1. **Имя проекта** — оставить `Hermes Dictate` или переименовать (например, `Hermes.Scribe`, `Dictate`)? Brand consistency.
2. **WhisperKit migration** — хотите сразу на нативный Swift (в 3-5x быстрее на M-чипах), или оставить faster-whisper?
3. **Глобальный hotkey** — какой chord? `Cmd+Shift+Space` (стандарт для dictation apps) или свой?
4. **Auto-paste** — должен ли включиться по умолчанию или opt-in (privacy concern)?
5. **iCloud sync vault** — нужно ли, или local-only strict?
6. **Obsidian integration** — открывать vault как Obsidian vault, или просто показать в Finder?
