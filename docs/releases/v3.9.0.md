# MyWhi 3.9.0 Public Preview

MyWhi is a local-first macOS dictation app powered by WhisperKit. This release is the first public preview prepared for GitHub distribution.

## What's New

- Native desktop shell with stable sidebar layout.
- Modern app icon and menu-bar icon.
- Russian / English localization that follows the macOS system language.
- WhisperKit model cache detection in Settings.
- Sparkle update integration with GitHub Releases.
- Markdown vault history, Scratchpad, Insights, and live recording HUD.

## Install

Download `MyWhi-3.9.0.dmg`, open it, and drag `MyWhi.app` into `Applications`.

This preview is ad-hoc signed and not notarized yet. If macOS blocks the first launch, right-click `MyWhi.app` and choose **Open**.

## Verification

- `swift test`: 147 tests, 3 skipped, 0 failures.
- `codesign --verify --verbose=2 dist/MyWhi.app`.
- `hdiutil verify dist/MyWhi-3.9.0.dmg`.

## Notes

Downloaded WhisperKit models are stored outside the app bundle in the persistent HuggingFace cache, so reinstalling MyWhi does not remove already downloaded models.
